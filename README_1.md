# Расчёт наценки на ювелирные изделия

Тестовое задание: анализ наценки на ювелирные изделия из золота и серебра за период с 01.04.2023 по 10.04.2023.

## Задача

По данным о продажах рассчитать среднюю наценку отдельно для изделий из золота и серебра.

Условия:
- период анализа: 01.04.2023 — 10.04.2023
- только продажи с суммой и себестоимостью > 0
- классификация металла по полю «Товарная группа»: группы на «СИ» — серебро, остальные — золото

## Результат

| Металл  | Средняя наценка |
|---------|----------------|
| Серебро | 214.73%        |
| Золото  | 118.45%        |

Наценка на серебро почти вдвое выше, что объясняется низкой себестоимостью серебряных изделий при сопоставимой розничной цене.
